# Portfolio Web Application

## Project Description
This is my personal portfolio web application, created to demonstrate my web development skills. The project includes the following sections:
- **Home** – main page with a brief introduction.
- **About** – information about me.
- **Projects** – list of projects with links to GitHub and Live Demo.
- **Skills** – technologies I use.
- **Contact** – contact information and contact form.

The project is built using Node.js and EJS for server-side rendering.

## Technologies
- Node.js
- Express.js
- EJS
- HTML / CSS / JavaScript
- Railway (hosting)
